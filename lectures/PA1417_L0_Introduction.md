## Scope & Learning outcomes
Software testing is a process of evaluating if software works as intended.
In this course you will learn to:
* Recognize, design, and follow the testing process
* Design and implement different types and levels of tests
* Use common testing tools
* Interpret test results
Full course syllabus - see Canvas page

## Course Scheduling
This is a mixed course for both Campus and Distance students.

**Lectures**
* We will present course materials, quizzes, Q & A
* Campus Students attend physically; Distance Students join via Zoom and can access recording

**Tutorials**
* Deep dive into course content directly relevant for the Labs. Live follow-along coding where possible. Detailed slides for later studying.

**Labs**
* Done in teams of 2 (Find your buddy ASAP)
* 2 labs with 3-4 assignments each (2x2.5 hp)
* Design and implement different types of tests
* Follow instructions, submit report on Canvas

**Online Q/A**
* Time to work on assignments in the team
* We will be available online for additional questions and help with the Labs (and any questions in general)

**Exam**
* Organized centrally by Tentamen office
* Campus Students: in person on paper; Distance Students: Online on Inspera
* Content based on the labs
* 2.5 hp, pass or fail
